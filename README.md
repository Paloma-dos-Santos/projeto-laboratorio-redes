## 📌 Descrição do Projeto

Este projeto tem como objetivo propor uma infraestrutura de redes de computadores para atender às necessidades do Grupo de Estudo e Pesquisa em Educação Ambiental de Sergipe, vinculado à Universidade Federal de Sergipe (UFS).

A proposta visa oferecer suporte tecnológico adequado às atividades de pesquisa, ensino e extensão desenvolvidas pelo grupo, garantindo conectividade, segurança, armazenamento de dados, hospedagem de aplicações e integração ao condomínio de laboratórios da instituição.

O projeto busca contribuir para o desenvolvimento e a divulgação de pesquisas acadêmicas na área de educação ambiental e ciências humanas.

---

## 🎯 Objetivos

- Planejar uma infraestrutura de rede escalável
- Garantir segmentação e segurança da rede
- Disponibilizar serviços de hospedagem institucional
- Permitir integração entre usuários, servidores e dispositivos IoT
- Organizar o tráfego da rede através de VLANs

---

## 🧠 Estrutura da Rede

### Componentes Utilizados

- Switch Layer 3 (Core)
- Switch Layer 2 (Acesso)
- Servidores de aplicação
- Servidor Web
- Banco de Dados
- Sistema de autenticação
- Access Points Wi-Fi
- Estações de trabalho
- Dispositivos IoT

## 🗂️ VLANs Implementadas

|---|---|

| VLAN 40 | Dispositivos IoT |
